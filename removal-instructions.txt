STEP-BY-STEP REMOVAL PROCESS:
================================================================================

STEP 1: Access Google Search Console
------------------------------------
1. Open browser and go to: https://search.google.com/search-console
2. Sign in with your Google account
3. Select property: freehillcapital.com (or www.freehillcapital.com)

STEP 2: Navigate to Removals Section
------------------------------------
1. Look at the left sidebar menu
2. Click on "Removals" (under "Indexing" section)
3. You'll see the URL Removal requests page

STEP 3: Request Removal for First URL
------------------------------------
1. Click the blue "New Request" button (top right)
2. Select "Temporarily remove URL"
3. Enter the full URL:
   https://www.freehillcapital.com/insights/the-importance-of-supplements-in-insurance-claims-for-contractors
4. Select option: "Remove this URL only"
5. Click "Next"
6. Review the URL to be removed
7. Click "Submit"
8. Wait for confirmation message

STEP 4: Request Removal for Second URL
------------------------------------
1. Click "New Request" button again
2. Select "Temporarily remove URL"
3. Enter the full URL:
   https://www.freehillcapital.com/industries-categories/healthcare
4. Select option: "Remove this URL only"
5. Click "Next"
6. Review the URL to be removed
7. Click "Submit"
8. Wait for confirmation message

STEP 5: Monitor Removal Status
------------------------------------
1. Stay on the "Removals" page
2. Check the status column for both URLs:
   - "Pending" = Request is being processed (up to 24 hours)
   - "Temporarily removed" = Successfully removed (lasts ~6 months)
   - "Cancelled" or "Failed" = Need to resubmit
3. Take screenshots of successful removals for records

STEP 6: Verify 301 Redirects Are Working
------------------------------------
1. Open URL Inspection tool (top of Search Console)
2. Enter first old URL and press Enter
3. Check that it shows:
   - Status: "URL is on Google" or "Redirect"
   - Redirect target: /about
   - Redirect type: 301 (Permanent)
4. Repeat for second URL
5. If redirects aren't showing, wait 24 hours for Google to recrawl

STEP 7: Optional - Test URLs Manually
------------------------------------
1. Open new browser tab
2. Paste first old URL: 
   https://www.freehillcapital.com/insights/the-importance-of-supplements-in-insurance-claims-for-contractors
3. Press Enter - should redirect to /about page
4. Open browser DevTools (F12 or Cmd+Option+I)
5. Go to Network tab
6. Reload page
7. Check first request shows "301" status code
8. Repeat for second URL
